# GL_MAX_TEXTURE_SIZE

## How to use

1. Download jar from Release
2. 
    ```shell
    $ java -jar GL_MAX_TEXTURE_SIZE.jar
    ```
   
## How to build

```shell
$ ./gradlew build
```
Then artifact built in ./build/libs/
